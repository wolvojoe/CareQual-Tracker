# CareQual-Tracker

![CareQual-Tracker Logo](https://raw.githubusercontent.com/wolvojoe/CareQual-Tracker/refs/heads/main/CareQual-Tracker.Web/Logos/CareQualLogo-Tran.jpg)


## Task 
1.  Allow care home records to be created. Dataset must contain as a minimum ‘Home Name’ 
2.  Allows care staff records to be added, edited and removed to care homes. Data set must 
contain as a minimum Forename, Surname and Date of Birth, Job Title, Annual Salary 
3.  Allow care qualification records to be added to staff records – add, edit end delete 
functionality should be implemented. Dataset must contain as a minimum qualification type, 
grade and attainment date 


## Built With

C# Web Forms N-tier Monolith
AdminLTE 4 Bootstrap Template
Entity Framework - Code First
Ninject
